# Payroll Mock Data

Monthly payroll accruals for October 2025. Files align with canonical balance + transaction fields so you can map them directly.

| File | Notes |
| --- | --- |
| `payroll_balances_balanced.csv` | Salaries payable + bonus accrual tie to GL 22100/22300. |
| `payroll_balances_variance.csv` | Missed bonus accrual causes a $60k understatement. |
| `payroll_register_variance.csv` | Detail register highlighting the missing bonus line.
